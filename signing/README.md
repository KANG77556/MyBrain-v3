# MyBrain AI 개인 서명키 보관 폴더

## 파일 역할

이 폴더는 개인 PC에서 `mybrain-release.jks` 같은 Android 서명키를 임시 보관할 때 사용합니다.

## 주요 기능

- `.gitignore`가 이 폴더 안의 개인 파일을 Git 추적 대상에서 제외합니다.
- 서명키와 비밀번호가 공개 GitHub 저장소에 올라가는 사고를 줄입니다.

## 수정 가능 영역

개인 PC에서는 이 폴더 안에 다음 파일을 넣어도 됩니다.

```text
mybrain-release.jks
개인 메모.txt
```

## 주의 사항

- 서명키 파일을 GitHub에 직접 올리지 마세요.
- 서명키 비밀번호를 소스 코드나 문서에 적지 마세요.
- 이 폴더의 `.gitignore` 파일은 삭제하지 마세요.
- 서명키는 암호화된 USB나 개인 클라우드에도 별도로 보관하세요.

## 변경 내역

고정 서명 자동화 단계에서 개인 서명 파일의 공개 업로드 방지를 위해 추가했습니다.
